Privacy Policy

DIAL transmits several types of information when necessary, all of which are used only to provide features to the user and not for tracking or analytics purposes.

If the Sidebar is enabled:

If you dont want your location to be used at all please disable the Sidebar option.

If the option to use "force weather coordinate" is not enabled, DIAL sends the IP address from which the request was sent to a service provided by ipinfo.io in order to estimate the user's location; this estimate is then transmitted to Dark Sky and displays the weather in your location. If you dont want your IP Address to be used, enable the coordinate option instead.

If the option is set to use "force weather coordinate" for weather is enabled, DIAL uses the given values to determine the users location. After the location is set DIAL sends the coordinates to Dark Sky and displays the weather conditions in your location.

Once the user's location has been estimated or determined by the above means, DIAL will send the information to a service provided by Dark Sky to retrieve the relevant weather conditions.
